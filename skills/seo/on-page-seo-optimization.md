# On-Page SEO Optimization

On-page SEO refers to the practice of optimizing individual web pages to rank higher in search engines and earn more relevant traffic. Unlike off-page SEO, on-page factors are entirely within your control.

## Learning Objectives

By the end of this skill, you will be able to:
- Optimize title tags and meta descriptions effectively
- Structure content using proper heading hierarchy
- Implement keyword placement best practices
- Optimize images for search engines
- Improve page load speed and user experience signals

## Core Concepts

### Title Tags

The title tag is one of the most important on-page SEO elements. It appears in search results as the clickable headline.

**Best practices:**
- Keep titles between 50-60 characters to avoid truncation
- Include your primary keyword near the beginning
- Make each title unique across your site
- Write for humans first, search engines second

**Example:**
```
Bad:  "Home | My Website"
Good: "Keyword Research Guide for Beginners (2024) | MarketingSkills"
```

### Meta Descriptions

While not a direct ranking factor, meta descriptions influence click-through rates from search results.

**Best practices:**
- Aim for 150-160 characters
- Include a clear call-to-action
- Naturally incorporate your target keyword
- Accurately describe the page content

### Heading Structure (H1-H6)

Headings help both users and search engines understand your content hierarchy.

- **H1**: Use once per page, include primary keyword
- **H2**: Main section headers, include secondary keywords
- **H3-H6**: Subsections, supporting topics

### Keyword Placement

Strategic keyword placement signals relevance to search engines without keyword stuffing.

**Key locations:**
1. Title tag
2. First 100 words of content
3. At least one H2 heading
4. Image alt text
5. URL slug
6. Meta description

### URL Structure

Clean, descriptive URLs improve both user experience and SEO.

```
Bad:  /page?id=123&category=seo
Good: /seo/on-page-optimization-guide
```

### Image Optimization

- Use descriptive file names (e.g., `on-page-seo-checklist.png`)
- Write meaningful alt text that describes the image
- Compress images to reduce file size
- Use modern formats like WebP where supported

## Practical Checklist

Use this checklist when optimizing any page:

- [ ] Primary keyword identified and researched
- [ ] Title tag optimized (50-60 chars, keyword included)
- [ ] Meta description written (150-160 chars)
- [ ] Single H1 tag with primary keyword
- [ ] Logical heading hierarchy (H2s, H3s)
- [ ] Keyword in first paragraph
- [ ] Internal links to related content (2-5 links)
- [ ] External links to authoritative sources
- [ ] All images have descriptive alt text
- [ ] URL is clean and descriptive (no query strings or unnecessary parameters)
- [ ] Page loads in under 3 seconds (check with PageSpeed Insights)
- [ ] Mobile-friendly layout verified
